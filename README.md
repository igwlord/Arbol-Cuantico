## 🚀 Instalación Rápida

```bash
# Clonar el repositorio
git clone https://github.com/igwlord/Arbol-Cuantico.git

# Navegar al directorio
cd Arbol-Cuantico

# Instalar dependencias
npm install

# Ejecutar la aplicación
npm run dev
```

La aplicación estará disponible en `http://localhost:5173/`
